# Dashboard de validacion de facturas KIE (Streamlit)

Tablero analitico del proyecto de grado "Sistema analítico de facturas para verificación normativa usando machine learning".

## Contenido del repositorio

- `dashboard_facturas.py` — aplicacion de Streamlit (archivo principal).
- `resultados_facturas_dashboard.json` — datos de todas las metricas y graficos.
- `db_facturas.csv` — base de referencia usada por el EDA 1.
- `requirements.txt` — dependencias.
